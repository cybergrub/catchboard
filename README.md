# CatchBoard - Fishing Competition Management

A modern web app for managing fishing competitions in Australia.

## Features

- **Public Side**: Browse events, pre-register, view leaderboards
- **Organiser Side**: Create events, manage registrations, verify catches, award prizes
- **Real-time Leaderboard**: Live updates as catches are submitted
- **QR Code Flyers**: Generate promotional materials
- **Photo Verification**: Event codes for catch authentication

## Tech Stack

- React 18
- Vite
- Tailwind CSS (via CDN)
- Lucide Icons

## Development

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This app is deployed on Vercel. Every push to main branch auto-deploys.
